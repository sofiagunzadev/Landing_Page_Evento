# Visão Geral do Projeto

O NEXORA Summit é uma landing page fictícia criada para representar um grande evento de inovação e negócios em Angola-Luanda.

Este projeto foi desenvolvido com foco em:

Alta conversão (inscrição em evento)
Clareza de informação
Design moderno e responsivo
Experiência de utilizador fluida (UX/UI)

---
# Objetivo

Demonstrar competências práticas em:

Estruturação de Landing Pages de alta conversão

UI/UX Design orientado a eventos

Desenvolvimento Frontend com HTML, CSS e JavaScript

Copywriting persuasivo para marketing digital

Organização de informação para geração de leads
---
# Problema Resolvido

Eventos de grande escala frequentemente perdem potenciais participantes devido a:

Informação confusa ou dispersa

Falta de foco em conversão

Experiência de utilizador fraca em dispositivos móveis


O NEXORA Summit Landing Page resolve isso ao centralizar toda a informação essencial num fluxo claro e persuasivo.
---

# Estrutura da Landing Page

A página foi desenhada com as seguintes secções:

1. Hero Section

Título impactante do evento

Subtítulo com proposta de valor

CTA principal (Inscrever-se)


2. Sobre o Evento

Descrição do propósito do summit

Público-alvo

Contexto de inovação e networking

🔹Benefícios

Networking com líderes da indústria

Acesso a tendências de tecnologia

Oportunidades de negócios

3. Agenda do Evento

Cronograma dividido por dias/horas

Destaque para keynote sessions e workshops

4. Expositores e Parceiros
 
5. Call to Action Final

Reforço de urgência

Botão de inscrição


6. Footer

Informações do evento

Redes sociais fictícias

---

# Decisões de UI/UX

Design minimalista com foco em conversão

Hierarquia visual clara (tipografia e espaçamento)

Uso de seções curtas para melhor leitura

CTAs repetidos estrategicamente ao longo da página

Mobile-first design

Contraste forte para destacar ações principais

---

# Tecnologias Utilizadas (Versão 1)

HTML5

JavaScript Vanilla


Nota: Futuras versões podem incluir React, TypeScript e integração com backend para inscrições reais.

# Estratégia de Marketing Simulada

A landing page foi pensada como se fosse um produto real:

Posicionamento: Evento premium de tecnologia e negócios em África

Público-alvo: Empreendedores, developers, investidores e estudantes

Gatilhos utilizados:

Escassez (vagas limitadas)

Autoridade (speakers e especialistas)

Oportunidade (networking e crescimento profissional)
